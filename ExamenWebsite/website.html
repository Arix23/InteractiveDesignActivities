Interview
Who is the End User for the Website?
People who want to know a little more about website creation, and see the process behind it.

What colors do you want for the Website?
Colors like black and white, that contrast, using more vibrant colors to highlight the important parts

Do you want a navigation bar?
Yes, so the user can travel between the two pages, the one talking about the process of creating the website and the main page.

How do you want the website creation page to look like?
I want it to look like a photo gallery, where people can see visually the process of creation.

Do you want images in the website?
Yes, mainly to help the user understand the information.
